Game Rules

Objective: Memorize and repeat the color sequence. Reach Round 20 to win!

How to Play:

  1. Watch the sequence of colored buttons light up
  2. Wait until the sequence finishes playing
  3. Click the buttons in the same order
  4. Each round adds one more color to remember
  5. Complete all 20 rounds to win the game!

Game Features:

  • Progressive Difficulty: Speed increases by 3% each round (max 100%)
  • Best Score: Your highest round is tracked during the session
  • Auto-Restart: Game restarts automatically if you make a mistake

Tips:

  • Focus on the pattern, not individual colors
  • Use visual memory techniques
  • Stay calm as the speed increases!
